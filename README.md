# Erdemozkan Tap

## How do I install these formulae?

`brew install erdemozkan/tap/<formula>`

Or `brew tap erdemozkan/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "erdemozkan/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
